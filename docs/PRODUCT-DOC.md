## 一、域名注册结论

### 1.1 品牌与推荐域名

| 项目     | 结论 |
|----------|------|
| **产品名** | **ClawHub**（与 CLI 工具名一致） |
| **首选域名** | **useclawhub.com**（已确认 WHOIS 无记录，可注册） |
| **备选域名** | clawhub.space、clawhub.plus（WHOIS 均返回 No match，可注册） |

**推荐理由**：`useclawhub.com` 符合「use + 产品名」的常见模式（如 useVercel、useSWR），易记且面向国际开发者。

## 二、域名注册商对比结论

### 2.1 推荐选择

| 场景 | 推荐注册商 | 理由 |
|------|------------|------|
| **国际化产品（主站）** | **Namecheap** | 免费隐私保护、价格透明、续费涨幅温和、对开发者友好 |
| **需要国内备案** | **阿里云（万网）** | 续费最稳、备案流程成熟、DNS 与云产品集成好 |
| **已在用腾讯云** | 腾讯云 | 可与 DNSPod、COS 等打通；注意续费比阿里云贵 |

### 2.2 价格与注意点（.com 为例）

| 注册商 | 首年 | 续费/年 | 隐私保护 | 注意 |
|--------|------|---------|----------|------|
| Namecheap | ~$8.88 | ~$15.98 | 免费 | 续费约涨 80%，可接受 |
| GoDaddy | 可低至 $0.01 | ~$17.99 | 付费 | 续费涨幅极大，不推荐长期持有 |
| 阿里云 | ¥83（新用户可¥1） | ¥90 | 部分免费 | 续费仅涨约 8%，长期成本低 |
| 腾讯云 | ¥88（新用户可¥1） | ¥148 | 约¥50/年 | 续费明显高于阿里云 |
| 火山引擎 | 需询价 | 需询价 | 支持 | 适合已用火山/字节生态的团队 |

### 2.3 不建议

- **GoDaddy**：首年便宜但续费暴涨，且默认自动续费、upsell 多，长期不推荐。
- **火山引擎**：域名非主营，价格不透明、资料少，除非强依赖字节生态再考虑。

## 三、产品定位（简要）

- **ClawHub**：AI 智能体的技能包（AgentSkills）注册中心 / 市场。
- 核心能力：发布与版本管理（类似 npm）、语义/向量搜索、一键安装（`npx clawhub@latest install <pkg>`）。
- 目标用户：全球开发者与 AI 应用构建者。
- 官网风格：北欧极简、英文为主、国际化（见 `demo3-nordic-minimal.html`）。

## 四、不同 TLD 的「权重」与区别

> 结论：**搜索引擎本身不按 TLD 给权重**，但不同后缀在**用户信任度、记忆成本、价格**上差异明显，会间接影响点击率和品牌。

### 4.1 搜索引擎（SEO）层面

| 问题 | 结论 |
|------|------|
| 不同 TLD 排名权重是否一样？ | **一样**。Google 等主流引擎对所有 gTLD 一视同仁，TLD 不是排序因子。 |
| 真正影响排名的是？ | 内容质量、外链、用户体验、站点权威度等，与用 .com 还是 .xyz 无关。 |

所以从「算法权重」角度，**.com / .io / .ai / .space / .xyz 没有区别**。

---

### 4.2 用户信任与行为（间接影响）

| TLD | 用户习惯与认知 | 说明 |
|-----|----------------|------|
| **.com** | 默认首选、信任度最高 | 约 43% 全球域名、用户猜网址时 57% 会先试 .com；信誉调查中得分最高。 |
| **.io** | 科技/初创/开发者认可 | 在技术圈接受度高，非技术用户相对陌生。 |
| **.ai** | 与 AI 产品强绑定 | 适合 AI 品牌，认知清晰；续费通常较贵。 |
| **.dev** | 开发者向 | Google 运营，技术产品常用；需验证开发者身份才能注册。 |
| **.xyz** | 新潮、通用 | 有知名案例（如 abc.xyz），但易被与廉价/ spam 联想，部分邮件/平台过滤更严。 |
| **.space / .plus / .tools 等** | 新 gTLD、小众 | 可用性高、有语义；大众信任度不如 .com，部分新 gTLD 滥用率较高（如 .top），.space 相对中等。 |

信任度会直接影响**点击率、口播传播、邮件到达率**，从而间接影响流量与转化。

---

### 4.3 按用途的简单建议

| 目标 | 更合适的 TLD |
|------|--------------|
| 最大信任度、泛用、长期品牌 | **.com**（若不可用，可用 useclawhub.com 等变体） |
| 面向开发者、技术品牌 | .com &gt; .io ≈ .dev |
| 明确 AI 产品形象 | .ai 有加分，但通常贵且可选少 |
| 先占位、低成本 | .space、.plus 等新 gTLD 可选，需接受信任度与记忆成本略低 |

---

### 4.4 与 ClawHub 的对应

- **主站/主品牌**：优先 **useclawhub.com**（.com + 「use」模式），兼顾信任与记忆。
- **备选**：clawhub.space / clawhub.plus 可作为次要或实验用，不建议作为唯一主域名。
- 不必为了「SEO 权重」去抢 .com；**内容与产品做好，比 TLD 重要得多**。
